# zerodiffusion
A tiny AI image detection model for Stable Diffusion, with an impressive size-to-accuracy ratio. It can run locally on most consumer grade GPUs, such as the NVIDIA Geforce GTX 1050 Ti.

<img width="1024" height="512" alt="image" src="https://github.com/user-attachments/assets/47737943-6312-4ee8-8375-7125fd32dd9c" />

## Usage
Ensure you have PyTorch and HuggingFace Diffusers installed on your system. Clone the GitHub repo and run `main.py`. It will automatically download needed model files.
